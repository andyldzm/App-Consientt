#  API Consientt – Gestión de Servicios de Pedicure

API desarrollada con **FastAPI** para gestionar un sistema de servicios podológicos como:

- Pedicure clínico
- Pedicure spa
- Uñas y esmaltado en gel
- Agenda de citas
- Registro de clientes

El sistema permite administrar clientes, servicios y citas mediante endpoints REST.

---

#  Tecnologías utilizadas

- Python
- FastAPI
- SQLAlchemy
- Uvicorn
- SQLite
- Redis (para cache)
